# Sky Rush

A fast-paced 3-lane endless runner built in Unity.

## Gameplay

- Control a ball speeding down a track that gets faster over time
- Switch lanes with **Left/Right arrow keys** (or swipe on mobile)
- **Jump** with Space (or tap on mobile)
- Dodge obstacles, collect coins, and survive as long as possible

## Project Structure

| Folder | Contents |
|--------|----------|
| `Assets/Scripts/` | All game logic (ball, obstacles, scoring, UI) |
| `Assets/Scenes/` | Unity scenes |
| `ProjectSettings/` | Unity project configuration |

## Built With

- Unity (C#)
- TextMesh Pro
- Unity Input System
